# Fitness Tracker API 

A backend API for tracking workouts (Bench Press, Squats, etc.) using Python and FastAPI.

## Features
* **Create:** Log new workouts.
* **Read:** View workout history.
* **Update:** Edit weight or reps.
* **Delete:** Remove entries.

## How to Run
1.  **Install:** `pip install fastapi uvicorn sqlalchemy`
2.  **Run:** `uvicorn main:app --reload`
3.  **View:** Go to `http://127.0.0.1:8000/docs`
